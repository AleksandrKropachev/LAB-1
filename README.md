# Задание 1

## Задача 1

### Текст задачи

Сформировать список из значений 1 и 0 по принципу: 0 — если очередное введенное число
нечетное, 1 — в противном случае.

### Алгоритм решения

функция принимает значение, сравнивает с нулем, если равна нулю, то выводит список, если нет то создает новый список, происходит проверка остатка от двух, если она равна нулю, то к списку присоединяется один, если не равно то присоединяется нуль и происходит перевызов функции с новым списком, до момента пока не введут нуль.

## Задача 2

### Текст задачи

Сформировать список из четных цифр числа

### Алгоритм решения

1 функция принимает число если оно равно нулю, то возвращает 0, если то отправляет число в модуле в другую функция создавая список, во 2 функции если число равняется нулю выводит результат(ноль для проверки прохода числа до конца), иначе создаем переменную с последней цифрой числа и создаем переменную после удаления последней цифры из числа, если остаток последней цифры после деления на 2 равен 0, то перевызывается функция с числом без последней цифры, а цифра присоединяется к результату, в ином случае мы просто перевызываем функцию без последней цифры, когда число кончится выйдет результат.
